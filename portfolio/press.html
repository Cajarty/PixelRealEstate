The PixelProperty ecosystem is powered by two types of assets, properties and tokens. The properties are cryptocollectables, which are tradable, collectable, digital property. Where each cryptocurrency token is the same as the next (that is, each Bitcoin is is just like any other Bitcoin), each digital property is a unique, non-fungible token. These properties cannot be created, and each represent a different region on the digital real estate market. The token of the PixelPropert ecosystem is a cryptocurrency known as PixelPropertyCoin which is required for certain interactions with the system.

Each piece of digital real estate can be modified to change its appearance, allowing for property owners to customize their desired appearance, as well as change the hover text and clickable link of each property. This means that properties can be used to showcase art, advertise websites, or share images.

Each piece of property can also be shared with the world, regular users to evolve the canvas as they please. PixelPropertyCoins are disbursed to users and property owners on the network through this shared usage.

Users do not need to own property to participate. Every property which is not yet owned can be changed by everyone, as well as every property that the owner shares can be as well. Regular users can earn PixelPropertyCoins by actively changing the canvas, contributing to the evolving digital canvas.
